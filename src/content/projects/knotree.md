KnoTree is a calm, botanical knowledge explorer that visualizes learning as a living tree. Start with a big topic, let the AI grow branches, and attach real sources as leaves. Users can prune distractions, deepen a single source into subtopics, and export a clean study outline.

## Highlights

- Bottom‑up tree growth that feels organic and readable.
- AI‑generated subtopics for fast topic decomposition.
- Source search with curated leaf nodes for manual selection.
- Prune, restore, and export a clean learning path.
- Botanical, paper‑texture UI designed for long-form reading.
